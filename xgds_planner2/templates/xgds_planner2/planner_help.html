Planner Help

The XGDS Planner has specific requirements for end-users in order to function properly.

The planner is known to work in Google Chrome. Make sure you're on the latest version by typing "about:help" in the URL bar.

Most of the functionality in the planner is dependent on Javascript. Javascript must be enabled in order for the planner to work properly.

The Google Earth Plugin is required to edit plans in any significant way. If you have an old version of the plugin and the planner doesn't work, you may need to uninstall it manually and re-install it to make it work.
