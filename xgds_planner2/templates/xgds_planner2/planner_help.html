Planner Platform Requirements

  • Browser: Google Chrome.

    Download Google Chrome. The planner is known to work in Chrome. It might work in other browsers, but if you run into problems we will ask you to switch to Chrome. (Sorry! With a small developer team it's difficult to support multiple browsers.)

  • Browser version: Latest.

    Google Chrome is designed to update frequently. To check that you are running the latest version, type about:help in the URL bar. If it isn't up to date, please click the update button (on the same page) and restart your browser.

  • Google Earth Plugin: Version 7 or greater.

    • Visit the version detector page and read the version of Google Earth Plugin in the yellow bar underneath the embedded viewer. (The version number will not appear until the page finishes loading, which may take a moment.)
    • If you have an version older than 7, you must uninstall it in order to install a newer version.
      • Windows: Select Start > Programs (or All Programs) > Google Earth > Uninstall Google Earth Plug-in, then restart your browser.
      • Mac: Delete /Library/Internet Plug-ins/Google Earth Web Plug-in.plugin, then restart your browser.
    • To install the latest version, visit the the version detector page again and it will guide you through installation.

  • JavaScript: Enabled.

    Most modern web pages require JavaScript, and you probably have JavaScript enabled already. This page detects if JavaScript is enabled in Chrome and provides instructions on how to enable it.
