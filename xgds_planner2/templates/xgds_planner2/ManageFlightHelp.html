Manage Schedule Help

A "Flight" is the execution of one or more plans at a given time on a given vehicle. For example, you may have a series of related plans that you intend to execute in a sequence on a certain day.

Schedule Plans:

  1. Go to the List Plans page.
  2. Check the box to the left of the plan you want to schedule.
  3. Click the Schedule Selected blue button on the upper right; this dialog will pop up:
  4. Click the box to the right of the When label and select the date AND time (24 hour clock) when you intend to run the plan. Note it does not have to be exact.
  5. Click the Schedule button
  6. You will notice that there is now a time in the Scheduled column on the list plans page.
  7. You can schedule the same plan to be run at different times.
Notes: behind the scenes, this created a flight for you if there was not already a flight for that day.

Manage Schedule:

In order to start up the video infrastructure on xGDS, you need to start (and stop) flights. In order to have accurate times in Playbook you need to start (and stop) plans.
  1. Go to the Manage Schedule page.
  2. You will see a list of the flights, indented under the flights are the plans scheduled for those flights.
  3. If you are just managing flights for today, it's nice to check the "Today Only" box on the upper left.
  4. To start a flight, click the start link next to the flight you want to start. Active flights will be listed in green. You should do this when the rover is up and running and there is a track in xGDS.
  5. To start a plan, click the start link next to the plan you want to start. Active plans will be listed in green.
  6. When the plan is done (aborted or completed) click the stop button next to the plan. The plan will no longer be green and it will show the stop time.
  7. When the flight is done, click the stop button next to the flight. The flight will no longer be green and it will show the stop time.
  8. If you accidentally added plans you don't intend to run to the flight, you can delete them.

Multiple flights on one day:

There may be a time when we have multiple executions on one day, i.e. the rover powers down midday, you move to a different site, or take a lunch break. There is no limit to how many flights you can set up for one day. In this case, you will have to manually create all subsequent flights for a day. Our naming convention for flights is DATELETTER_VEHICLE, for example 20140912A_KRex2.

To create a flight manually:

  1. Go to the Manage Schedule page https://mvp.xgds.org/xgds_planner2/manage/
  2. Click the blue Create Flights button on the upper right.
  3. Choose the date of the flight you want to create.
  4. Choose the prefix letter of the flight; most likely it will be B or higher. Please use uppercase.
  5. Make sure the vehicle(s) you want are checked.
  6. Click Create.

To schedule plans with one of these hand-created flights, see the steps for Schedule Plans above.

  1. When you are filling out the dialog, choose the flight from the drop down (i.e. 20140912B_KRex2).
  2. You also must enter the date and time for the plan to execute.
  3. Press Schedule.
  4. On the Manage Schedule page you will see the plan scheduled under the flight you selected.
